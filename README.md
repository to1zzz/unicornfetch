## Installation

### From Source
Make sure you have Go installed, then clone the repo and install the binary to your system layout:

```bash
git clone [https://github.com/to1zzz/unicornfetch.git](https://github.com/to1zzz/unicornfetch.git)
cd unicornfetch
# Compile the binary
go build -o unicornfetch op.go
# Move it to your local bin directory so it can be called from anywhere
sudo cp unicornfetch /usr/local/bin/
